SIMPLE SHOWCASE

SIMPLE SHOWCASE is a very simple & clean Bootstrap HTML Template. 
It is meant for those with very basic needs of general website showcasing photos or creative works.

Clean landing page (easy to change header text & call to action buttons).
Sliding Header with flexible image ratio (no adjustment required).
Responsive Photo Lightbox
Bootstrap Dropdown with on hover behavior.
Responsive Video support.
Easy to change logo & slogan text for quick start your site.
Pinterest like gallery for showcasing photos or videos.
Sample Bootstrap snippets.
Sample Simple Page for you to start with your own content.
Custom style is separated from the original Bootstrap Framework (stored in a separate CSS file), so updating Bootstrap is as easy as 1-2-3!

Please enjoy!



TEMPLATE CONTENTS:

- index.html => Sample landing page

- page.html => Sample Bootstrap snippets (includes responsive videos, Awesome icons)

- grid.html => Sample media gallery with responsive lightbox

- simple.html => Sample simple page for you to start with your own content
